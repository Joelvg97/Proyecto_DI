# Teachers App

Aplicación creada con **React + Vite** para gestionar la asistencia de estudiantes.  
Permite marcar a cada alumno como **Presente**, **Ausente** o **Tarde**, actualizando las estadísticas en tiempo real.

---

## Cómo ejecutar el proyecto

### 1. Instalar dependencias

- cd teachers_app
- Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
- npm install

### 2. Ejecutar en modo desarrollo

- npm run dev

## Características

- Lista de estudiantes con foto e ID.
- Botones para marcar **Present / Absent / Late**.
- Estadísticas globales actualizadas en tiempo real.
- Interfaz responsive con **Tailwind CSS**. Adaptado a dispositivos móviles, tablets y pantallas grandes.
